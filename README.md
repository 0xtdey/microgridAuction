## Microgrid Auction

Clone this repository locally using the following command in vs code terminal
```shell
 git clone https://github.com/0xtdey/microgridAuction
```

## Usage

### To Build the project use :

```shell
 forge build
```

### To Test use :

```shell
 forge test
```
### To get coverage report use :

```shell
forge coverage
```
